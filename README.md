


### ✨ Quote Generator

A clean and responsive quote generator built with **HTML**, **CSS**, and **JavaScript**. It displays inspiring quotes with a modern interface and useful features.

**✨ Features**

* 🎲 Random inspirational quotes
* 📋 Copy quotes to clipboard
* 📤 Share quotes
* 🌙 Dark Mode
* 📱 Fully responsive design
